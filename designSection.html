    Creative Reflection:

    Looking at the first colour palette of the style guide. The first option was lilac, for the background which was a soft purple but I felt as if the soft purple with the black header was a bit unprofessional mixed with the first font and decided to go with a soft pink as the background. At first I experimented with dark red but the red was conflicting with the black font and changing the font into white might be a bit distracting to viewers so soft pink felt like a safe colour which would not conflict with the fonts of the website. This soft pink helped compliment the website with the neutral font colours such as black and white. I changed the text align to make the blog post section more neat and polished.
